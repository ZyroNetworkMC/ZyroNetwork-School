# Introduction to Nukkit Plugin Development

Welcome to the **ZyroNetwork School** guide on Nukkit plugin development! Nukkit is a highly efficient, Java-based server software for Minecraft: Bedrock Edition. Because it leverages the Java Virtual Machine (JVM), Nukkit is capable of incredible performance and allows developers to utilize the vast ecosystem of Java libraries.

## What is Nukkit?

Nukkit (and its popular fork, Cloudburst Nukkit) offers a robust API heavily inspired by the Bukkit API from Minecraft Java Edition. If you've ever made a plugin for Spigot or Bukkit, you will feel right at home with Nukkit!

### Why Choose Nukkit?
- **Java Power**: Take advantage of multi-threading, rich standard libraries, and modern Java features.
- **Performance**: Capable of handling hundreds of players on a single instance due to asynchronous architecture.
- **Familiar API**: Event listeners, command execution, and task scheduling mirror standard Bukkit/Spigot paradigms.

## Prerequisites

Before jumping into code, ensure you have the following:

1. **Java Development Kit (JDK) 17 or newer**: Nukkit runs on modern Java.
2. **An IDE**: IntelliJ IDEA, Eclipse, or VS Code (with Java extensions). We highly recommend IntelliJ IDEA.
3. **Maven or Gradle**: You will need a build tool to compile your plugin into a `.jar` file and manage dependencies.
