# What is an Operating System?
An Operating System acts as a bridge between:
```text
User
 ↓
Operating System
 ↓
Hardware
```
Example:
```text
You click Minecraft
      ↓
Windows/Linux handles everything
      ↓
CPU executes instructions
```
---
# Linux Architecture
```text
Applications
      ↓
Shell
      ↓
Kernel
      ↓

```

The kernel is the heart of Linux.

- Memory Management
- CPU Scheduling
- Device Drivers
- File Systems
- Networking
