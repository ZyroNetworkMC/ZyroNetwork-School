# Linux `cd` Command

The `cd` command is used to move between directories in Linux.

It is one of the most frequently used commands because Linux systems contain many folders and files, and navigation is impossible without `cd`.

---

# What does `cd` mean?

`cd` stands for:

```text
Change Directory
```

---

# Syntax

```bash
cd [DIRECTORY]
```

---

# Basic Usage

Suppose your current location is:

```text
/home/aayan
```

Move into Documents:

```bash
cd Documents
```

Check:

```bash
pwd
```

Output:

```text
/home/aayan/Documents
```

---

# Current Directory Structure

```text
/home/aayan
├── Documents
├── Downloads
├── Projects
└── Desktop
```

---

# Going to an Absolute Path

Absolute paths always start from:

```text
/
```

Example:

```bash
cd /etc
```

Check:

```bash
pwd
```

Output:

```text
/etc
```

---

Another example:

```bash
cd /var/log
```

Output:

```text
/var/log
```

---

# Relative Paths

Relative paths start from your current directory.

Current location:

```text
/home/aayan
```

Move:

```bash
cd Projects
```

Linux automatically interprets:

```text
/home/aayan/Projects
```

---

# Command Shortcuts

## `cd`

No argument.

Moves to home directory.

```bash
cd
```

Output:

```text
/home/aayan
```

---

## `cd ~`

Also moves to home directory.

```bash
cd ~
```

---

## `cd ..`

Move one directory backward.

Example:

Current:

```text
/home/aayan/Documents
```

Command:

```bash
cd ..
```

Result:

```text
/home/aayan
```

---

## `cd ../..`

Move two directories back.

Current:

```text
/home/aayan/Projects/Web
```

Command:

```bash
cd ../..
```

Result:

```text
/home/aayan
```

---

## `cd -`

Move back to previous directory.

Example:

```bash
cd /etc
cd /var/log
cd -
```

Output:

```text
/etc
```

---

Another:

```bash
cd -
```

Output:

```text
/var/log
```

This toggles between directories.

---

## `cd /`

Move to Linux root directory.

```bash
cd /
```

Output:

```text
/
```

---

## `cd .`

Current directory.

Nothing changes.

```bash
cd .
```

---

# Using Tab Completion

Suppose:

```text
Documents
Downloads
Desktop
```

Type:

```bash
cd Doc
```

Press:

```text
TAB
```

Linux automatically completes:

```bash
cd Documents
```

---

# Directory Names with Spaces

Wrong:

```bash
cd My Folder
```

Correct:

```bash
cd "My Folder"
```

or:

```bash
cd My\ Folder
```

# Common Mistakes

---

## Wrong Directory Name

```bash
cd document
```

Output:

```text
No such file or directory
```

Linux is case-sensitive.

Correct:

```bash
cd Documents
```

---

## Forgetting Spaces

Wrong:

```bash
cd/home
```

Correct:

```bash
cd /home
```

---

## Not Knowing Current Location

Always use:

```bash
pwd
```

before navigating.

# Flags / Special Arguments Summary

| Command | Description |
|----------|-------------|
| `cd` | Go home |
| `cd ~` | Home directory |
| `cd ..` | Parent directory |
| `cd ../..` | Two directories back |
| `cd -` | Previous directory |
| `cd /` | Root directory |
| `cd .` | Current directory |
